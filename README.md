## サービス概要
「ゲームを長時間やってしまう」、「ゲーム仲間に上手く切り出せない」を解決するアプリです。

## このサービスへの思い・作りたい理由
『自分がよくゲームをしていた時期に、夢中になってなかなかやめようと思えない』、『一緒にプレイしているゲーム仲間へ「やめます」と言いづらい』といった問題が発生していた。この問題が解決しやすくなれば、時間管理がスムーズに行きやすくなる。

## ユーザー層について
「楽しむときは楽しむ。やるべきことはしっかりやる。」というのを身につけ、時間を有効活用することの手助けとなるアプリを作りたいと思ったから。

## サービスの利用イメージ
ゲームを切り上げたいときの心理的障壁が減り、自分の思うように時間管理ができる。
##### <心理的障壁を減らすための工夫>
・ゲームの終了予定時刻から何分以内にやめることができたかを統計データで可視化することで、達成感を得られる。
・定型文機能により、ゲーム仲間にやめることをどう切り出せばいいかをその都度考える必要がないため、「どう伝えるか考えなきゃ…」と思う必要がなく、「この場合であれば、こんな感じで言おう」とスムーズに考えることができる。

## ユーザーの獲得について
・友達に使ってもらう。
・XやMattermostなどで公開して、使ってもらう。

## サービスの差別化ポイント・推しポイント
ゲームを中断することへのフォーカスがある。（ゲーム仲間にゲームを途中でやめることを伝える時の心理的障壁を減らすための工夫がある）

## 機能候補
### <MVPリリースまで>
・ゲームの時間設定機能（タイマー・アラーム）。
・決めた時間から何分以内にやめることができたかをグラフなどにして可視化。
・ゲームを途中でやめるときにゲーム仲間に言う定型文一覧（デフォルト & 自分で登録）（※デフォルトの定型文は、初期段階（MVP）では、理由の項目を3,4パターンにして、パターンごとに2,3個ずつ用意しておこうと考えています）
#### <デフォルトの定型文の例(理由別)>
##### <予定が急に入った場合>
・「申し訳ありません、急用が入ってしまったので今日はここで失礼します。」
・「急な用事ができちゃったので今日はここで抜けますね、ごめんね。」
##### <ゲームの進行が悪くて気まずくなった場合>
・「ごめん、今日はちょっと調子悪いので抜けます！また次回リベンジさせて…！」
・「集中できなくなってきたので、今日はここで抜けます！」

### <本リリースまで>
・googleカレンダー等と連携して、スケジュール確認をしやすくする。
・ゲーム日記機能（決められた時間でのゲームプレイに対する達成感を得るため）
・ゲーム仲間とタイマー・アラームを共有して、同時に切り上げができる機能

## 使用する技術スタック
・バックエンド：Ruby on Rails
・タイマー・アラーム：Stimulus
・タイマーの開始と、終了通知の同期：Action Cable
・スケジュール連携：Google Calender API
・グラフ：Chart.js
・データベース：PostgreSQL
・デプロイ先：Render
